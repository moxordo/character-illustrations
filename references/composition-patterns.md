# Composition Patterns and Originality Rules

## Structure types

Pick one structure per image; do not mix several.

### Workflow
For: input → processing → output, content production, AI workflows, automation chains.
Draw: input on the left, the character or a strange machine processing in the middle, output on the right, the flow accent marking the main direction.

### System slice
For: information sources, filters, databases, renderers, one corner of an agent system.
Draw: only 3–5 core modules; the character performs one key action inside one of them.

### Before / after
For: chaos/order, manual/automated, scattered/gathered, anxious/stable.
Draw: chaos left, stability right, one accent arrow between. The character may be more exaggerated here.

### Character state
For: user pain points, creator states, too-many-tools, information anxiety, stuck→running.
Draw: 2–4 small states, one short label each.

### Concept metaphor
For: content factory, information warehouse, black-box brain, workflow machine, auto-report.
Draw: one big strange object or machine, minimal inputs, one output. Must be memorable.

### Layered method
For: methodology frameworks, system levels, capability stacks, content-system layers.
Draw: stacked boxes, never a formal pyramid; the character hauls bricks or builds at the side.

### Map route
For: idea-to-launch, user journeys, hand-off paths, learning routes.
Draw: one winding path, few nodes, the character leading the line or walking it.

### Mini comic
For: fail-to-success, the real process, gentle complaint, before/after using something.
Draw: 2–4 small panels, one action per panel.

## Inventing a fresh metaphor (every time)

Reinvent the metaphor from the current content. Never re-serve an old image.

1. Turn the abstract concept into a **physical action**: stuck, leaking, getting heavier, sorting, settling, fermenting, opening a door, folding, unpacking, flowing back.
2. Turn the system structure into a **low-tech object**: broken machine, cardboard box, drawer, water pipe, mailbox, strange dial, scale, well, ladder, gate, turntable, black box, hole punch, pasta press, clothesline, odd workstation.
3. Give the character the **action**: not standing beside — stuck inside the machine, pulling the wrong line, guarding a gate, hauling, mending, weighing, holding the ladder, recording, stuffing things into a strange device.

Pools (pick 1–2 objects and 1 action; never pile up):
- Objects: box, drawer, old machine, funnel, scale, mailbox, door, well, ladder, pipe, ball of thread, gate, turntable, black box, punch, press, clothesline, workstation.
- Actions: pull, carry, stuff, scoop, press, weigh, sew, cut, twist, guard, push, catch, dismantle, mark, recycle.
- The action must serve the core idea — never strange for strange's sake.

## Multi-actor scenes

When a scene contains multiple actors (people, teams, agents, systems):

- **Label every actor with who they are** (e.g. `on-call`, `data team`, `AI agent`, `us`) — even if that pushes past the 5–8 label budget. An unlabeled actor is a QA failure.
- Show the acting party explicitly — if someone asks a question or triggers an event, draw *who*.
- When a mechanism matters, draw the mechanism concretely (a gauge, graduated cups and a balance scale — not a box labeled "measurement").

## Anti-copy rules

Old images (bundled examples, previous batches) calibrate only line density, whitespace, color restraint, and the character's demeanor. Do not reuse their compositions, objects, or annotations unless the user explicitly says "like this one / remix this".

Known worn-out compositions to avoid re-serving: conveyor belt with breakpoints, character pulling a judgment lever inside a machine, character-as-funnel sorting, cutting the material fish, leading the hand-off path, pulling three source lines, stamping a phrase toolbox, holding a sign over a pit path, an idea press.

Same theme still gets a new metaphor: a "hand-off path" need not be a route — the character can splice a content tail onto a door handle; "one fish many dishes" need not be a fish — the character can press one paper ball into several shapes.
